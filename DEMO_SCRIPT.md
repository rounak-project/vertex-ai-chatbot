# VERTEX Demo Script

## 2-3 Minute Speaking Script

Good morning respected teacher and my friends.

My project is called VERTEX, which means Space AI Assistant. I built it as a Class 7 school project to make learning about space more interesting and interactive.

I chose this idea because many students like space, rockets, planets, and astronauts. Instead of only reading facts from a book, VERTEX lets us ask questions, see space data, and explore a dashboard.

First, this is the chatbot section. I can ask VERTEX a space question, like "Tell me about Mars." The app first checks its local brain, which is a JSON knowledge file. This makes the project work even without internet.

I also added optional real AI support using the Groq API. If a Groq API key is added, VERTEX can answer extra questions using AI. If there is no key, the app still works in offline demo mode.

Next, this is the NASA Picture of the Day section. The Flask app asks NASA for a space image. If the NASA API is not available, VERTEX shows a backup image so the demo does not fail.

This is Mission Control. It shows the International Space Station location and a rocket launch dashboard. The ISS tracker can use live data, and the launches are shown from local demo data.

VERTEX also has planet cards, space agency cards, voice input, text-to-speech, a space quiz, theme switcher, and presentation mode. Presentation mode makes the screen cleaner for showing the project.

While making this project, I learned Python, Flask routes, APIs, JSON data, HTML, CSS, JavaScript, localStorage, and how a chatbot can use both local knowledge and optional AI.

In conclusion, VERTEX is not only a chatbot. It is a complete space learning dashboard. Thank you.
