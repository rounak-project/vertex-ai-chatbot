# VERTEX Demo Script

## 2-3 Minute Speaking Script

Good morning respected teacher and my friends.

My project is called VERTEX AI OS. It is an artificial intelligence and technology operating system that I built with Python, Flask, HTML, CSS, JavaScript, local JSON data, browser voice features, and optional Groq AI.

I created VERTEX to make advanced technology easier to explore. Instead of only reading about AI, programming, cloud, cybersecurity, robotics, and new tools, a student can open one dashboard and interact with many learning features.

First, this is Vertex Chat. I can ask a technology question like "What is generative AI?" The app first checks its local technology brain, which is stored in JSON and Python logic. This means the project still works even without an API key.

I also added optional real AI support using the Groq API. If a Groq API key is added in the environment, VERTEX can use an online model for harder technology questions. If there is no key, it automatically stays in offline local mode.

The dashboard also includes an AI news signal feed, an AI models index, AI companies, AI tools, technology trends, and an AI landscape explorer. These sections help users understand the modern AI ecosystem.

Next, VERTEX includes learning and productivity tools. The AI Academy quiz tests knowledge and calculates a score. The Website Builder helps create a site plan. The Coding Workspace creates starter project files. The Prompt Library stores useful prompts. The Roadmaps page helps track learning progress, the Whiteboard renders diagrams, and the Interviewer helps practice interview answers.

I also added theme switching, presentation mode, voice input, and text-to-speech. These features make the app feel more like a complete product instead of only a simple chatbot.

While making this project, I learned Flask routes, API design, JSON data, frontend interactivity, localStorage, responsive design, AI fallback logic, and how to prepare a project for deployment on Render.

In conclusion, VERTEX AI OS is a complete AI learning dashboard and technology assistant. Thank you.
