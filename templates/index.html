Final Summary

Project Summary

What Is Vertex?

VERTEX is a space-themed AI assistant dashboard made with Python, Flask, HTML, CSS, JavaScript, NASA data, and an optional Groq AI brain.

What Problem Does It Solve?

It makes space learning more interactive by combining chatbot answers, space facts, NASA images, mission data, and a quiz in one demo.

Features

  • Chatbot with local brain and optional AI API.
  • NASA image, ISS tracker, launches, planets, and agencies.
  • Voice input, speech output, themes, quiz, and presentation mode.

Technologies Used

  • Python, Flask, JSON, and environment variables.
  • HTML, CSS variables, JavaScript fetch, and browser APIs.
  • NASA API, ISS API, and optional Groq API.

What I Learned

I learned how a web app sends requests, loads data, handles errors, stores settings, and connects a simple chatbot to optional AI.

About Vertex

Built For A School Demo

What Vertex Is

VERTEX is a futuristic space-themed AI chatbot made as a beginner-friendly school project. It helps students ask questions and explore space facts.

What It Can Do

  • Answer simple space questions.
  • Show NASA's picture of the day.
  • Track the ISS with live or demo data.
  • Show planets, agencies, launches, and news.
  • Use voice input, speech output, themes, quiz mode, and presentation mode.

Technologies Used

  • Python and Flask for the web server.
  • HTML, CSS, and JavaScript for the dashboard.
  • JSON files for beginner-friendly data.
  • NASA APOD and ISS APIs with safe fallback data.

6-Day Roadmap

  1. Day 1: Project setup.
  2. Day 2: Chatbot brain.
  3. Day 3: NASA API and dashboard.
  4. Day 4: Mission Control, voice, and speech.
  5. Day 5: Futuristic assistant experience.
  6. Day 6: Final testing and presentation.

School project note: VERTEX is designed for learning and demos. It keeps the code simple, uses comments, and avoids heavy libraries so a Class 7 student can explain it.
